Bagaimana Internet Bekerja Secara Umum?

Internet adalah jaringan komputer global yang saling terhubung, memungkinkan perangkat di seluruh dunia untuk berkomunikasi dan berbagi informasi. Ini seperti sistem pos raksasa yang tidak memiliki pusat, di mana data dapat dikirim dari satu titik ke titik lain melalui berbagai jalur.

Bayangkan kamu ingin mengirim surat (data) ke teman di negara lain. Kamu tidak mengirimkannya langsung. Kamu memberikannya ke kantor pos lokal (router-mu), yang kemudian meneruskannya ke pusat sortir (ISP), yang kemudian mengirimkannya melalui pesawat atau truk (kabel fiber optik bawah laut) ke negara tujuan, lalu ke kantor pos lokal temanmu, dan akhirnya diantar ke rumahnya (komputernya).

Proses dari Awal sampai Akhir

Secara sangat sederhana, inilah yang terjadi saat kamu membuka website, misalnya google.com:

  1. Permintaan Dimulai (Browser): Kamu mengetik google.com. Browser-mu adalah klien yang memulai permintaan. (Dibahas di halaman Browser).
  2. Pencarian Alamat (DNS): Browser-mu tidak tahu di mana "rumah" google.com. Ia bertanya pada "buku telepon internet" (DNS) untuk menerjemahkan nama domain itu menjadi alamat IP (misal: 172.217.14.228).
  3. Mencari "Rumah" (Hosting): Setelah tahu alamat IP-nya, browser-mu tahu harus pergi ke server mana di dunia ini. Server ini adalah layanan Hosting, tempat file-file website google.com disimpan.
  4. Komunikasi (HTTP): Browser-mu mengirim permintaan "Tolong kirimkan saya halaman utama" menggunakan bahasa protokol HTTP (atau HTTPS yang aman).
  5. Respon & Tampilan: Server Hosting mengirimkan kembali file-file (HTML, CSS, JS) sebagai respon. Browser-mu menerima file-file ini dan "merendernya" menjadi halaman web yang kamu lihat di layar.
