Apa itu Browser?

Browser (atau Peramban Web) adalah perangkat lunak (software) yang berfungsi untuk menerima, mengakses, dan menyajikan sumber informasi di World Wide Web. Sederhananya, ini adalah aplikasi yang kamu gunakan untuk "membuka" internet, seperti Google Chrome, Mozilla Firefox, atau Safari.

Bagaimana Browser Bekerja?

Proses kerja browser bisa disederhanakan dalam beberapa langkah:

  1. Input Pengguna: Kamu mengetikkan alamat web (URL), misalnya www.google.com, di address bar.
  2. Pencarian DNS: Browser tidak tahu di mana google.com tinggal. Ia bertanya kepada "buku telepon internet" (DNS) untuk menerjemahkan nama domain itu menjadi alamat IP (misalnya 172.217.14.228).
  3. Membuat Permintaan (Request): Setelah tahu alamat IP servernya, browser mengirimkan permintaan HTTP (HTTP Request) ke server tersebut, meminta "Tolong kirimkan saya file untuk halaman google.com."
  4. Menerima Respon: Server Google menerima permintaan itu dan mengirimkan kembali file-file yang dibutuhkan (HTML, CSS, JavaScript) sebagai Respon HTTP (HTTP Response).
  5. Rendering: Ini adalah bagian ajaibnya. Browser membaca file HTML untuk memahami struktur halaman (di mana letak judul, paragraf, gambar). Kemudian, ia membaca file CSS untuk memahami styling (apa warnanya, bagaimana tata letaknya). Terakhir, ia menjalankan JavaScript untuk hal-hal interaktif (seperti animasi atau pop-up).
  6. Menampilkan Halaman: Browser "menggambar" semua elemen yang sudah di-render tadi ke layar kamu, sehingga kamu bisa melihat halaman web yang utuh dan interaktif.
